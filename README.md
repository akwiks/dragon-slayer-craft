# ⚔️ Dragon Slayer: Руководство Черного Мечника

> *"Это было слишком огромным, чтобы называться мечом. Толстое, тяжелое, грубое и заржавелое. Скорее это был кусок сырого железа."*

Добро пожаловать в кузницу! Этот проект представляет собой интерактивный веб-сайт — полное руководство по крафту легендарного меча Гатса из аниме и манги **«Берсерк»** своими руками из обычной деревянной доски.

## 🛠️ Особенности проекта
* **100% No-JS:** Весь интерактив (переключение вкладок, раскрывающиеся меню, анимации) реализован исключительно на чистом **HTML** и **CSS**.
* **Подробный гайд:** Пошаговая инженерно-крафтовая инструкция от разметки до финальной оплетки рукояти.
* **Академия боя:** Интерактивный разбор боевых техник Гатса с физикой инерции и траекторий.
* **Суровая поддержка:** Встроенный блок мотивации с цитатами Кратоса для тех, кто устал шкурить сосну.

## 📦 Как запустить локально
1. Склонируйте репозиторий или просто скачайте файл `берсерк.html`.
2. Откройте `берсерк.html` в любом современном браузере.
